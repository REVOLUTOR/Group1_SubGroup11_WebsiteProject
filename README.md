## Project Description

TechVision Solutions is a responsive two-page business website built entirely with HTML5 and CSS3, with no JavaScript or CSS frameworks.

- **Homepage (`index.html`):** header with navigation and call-to-action, hero section, about us, a 6-card services grid, an 8-image portfolio gallery, testimonials, a 3-tier pricing section, a contact form, and a footer.
- **Blog page (`blog.html`):** matching header and footer, a blog banner, a full-length article with headings, lists, and a blockquote, a sidebar with search, recent posts, and categories, an author card, related articles, a comments section, and a newsletter signup.

Layouts are built with CSS Flexbox (header, hero, pricing, blog layout) and CSS Grid (services, portfolio, related articles), with hover effects, transitions, and transforms used throughout. The site is responsive down to a `768px` breakpoint, with the blog sidebar moving below the main content and all grids stacking to a single column on smaller screens.
